# FreeCAD Weekly Updater :gear:

Updates FreeCAD weekly development version to local directory.

- Just one simple Python script.
- Runs on Python >= 3.11.
- No third-party libraries are used.
- Tested on Windows 11. 

## Usage :rocket:

- Copy the Python script to a local folder where you want the FreeCAD weekly version to be installed.
- Run the Python script. The latest FreeCAD development version will be downloaded and unzipped.
- When there is a new FreeCAD version, just run the Python script again.
- If you want some automation, run this script with a weekly cron job.
